## Dirty Data
###### A twitter and facebook data visualizer


#### Prequisites:
* [node](http://nodejs.org/)
* [mongodb](http://www.mongodb.org/downloads)


#### Install:
````shell
git clone git@github.com:CoreyTrombley/dirty_data.git

npm install
````


#### Starting up:
````shell
node app
````


#### Routes:
* "/" - login page
* "/dashboard" - data visualization page
* "/ff" - users api json
